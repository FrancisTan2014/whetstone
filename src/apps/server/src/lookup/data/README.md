# Bundled lookup data

## `cedict.u8.gz` — CC-CEDICT

`cedict.u8.gz` is the gzip-compressed [CC-CEDICT](https://www.mdbg.net/chinese/dictionary?page=cc-cedict)
community Chinese-English dictionary, used by the Chinese vocabulary-lookup provider. It is
decompressed and parsed into an in-memory index at server startup.

- **Source:** MDBG — https://www.mdbg.net/chinese/export/cedict/
- **License:** Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0) —
  https://creativecommons.org/licenses/by-sa/4.0/
- **Referenced work:** CEDICT — Copyright (C) 1997, 1998 Paul Andrew Denisowski
- **Publisher:** MDBG. Bundled version: 1.0 (format `ts`, UTF-8).

Per CC BY-SA 4.0, this attribution is retained here, the reader surfaces a CC-CEDICT
attribution alongside Chinese results, and any modifications to the dataset would be shared
under the same license. The file is bundled verbatim (gzipped); it is not modified.
